**Hosts** are any devices which sends or receives traffic :

some common examples include ->
- Computer/Laptops
- Phones
- Printers/Scanners
- Servers
- Cloud Computing/Cloud Servers

- Also any [[Internet of Things (IoT)]] devices
  which include 
- TV
- speakers 
- refrigerators
- smartwatches
- lights
- any digital gadget these days mostly is IoT enabled
